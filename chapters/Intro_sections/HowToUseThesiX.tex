\section{How to Use \textit{ThesiX}}
\ThesiX{} have the following structure:

\begin{lstlisting}
\documentclass[a4paper,11pt,openright]{book}
\input{settings/define_basic} % Definition of basic info or LaTeX environments
\input{settings/define_shortcut} % Definition of useful word shortcuts
\input{settings/packages} % Packages and settings
\input{settings/acronyms} % Acronyms definition
\input{settings/symbols} % Symbols definition
%-------------------------------------------------------
\begin{document}
\pagenumbering{gobble} %No page numbers
\input{covers/title}
\input{covers/titleb}
\blankpage
\input{covers/tribunal}
\blankpage
\input{covers/dedication}
\blankpage
\input{covers/phrase}
\blankpage
\input{settings/coverstyle}
\input{covers/acknowledgements}
\frontmatter
\input{covers/resumen}
\input{covers/abstract}
\input{covers/tableofcontents}
\mainmatter
\input{settings/chapterstyle}
\include{chapters/chapter_Intro}
\include{chapters/chapter_Elements}
%-----------------------------------------------------------------------
\input{settings/coverstyle}
\appendix
\input{appendix/appendix_Example}
%%----------------------------------------------------------------------
\input{bibliography/bibliography}
\input{extrainfo/extrainfo}
%%----------------------------------------------------------------------
\cleardoublepage  % Ensure finishing the last section on an even page
\pagestyle{empty} % Remove page number, headers, etc.
\blankpage        % Add two blankpages to add a final blank paper sheet
\blankpage
\end{document}
\end{lstlisting}
